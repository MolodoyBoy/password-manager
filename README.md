# Password Manager

## Tech, Frameworks and other Dependencies

* Java version: Eclipse Temurin 21
* Gradle version: 8.6
* Spring Boot version: 3.2.3
* Postgres 15
* Docker

## Launch application

* Start database
```shell
docker compose up
```
* Build project 
```shell
./gradlew build
```
* Run project
```shell
./gradlew :bootRun
```
